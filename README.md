<h2>Introduction</h2>

A collective todolist for business built with Angular 17.

<h2>Features</h2>
The application is a website that can organise collective todolist.
Users can do the following :
<ul>
    <li>Consult list of todos</li>
    <li>Create todo</li>
    <li>Remove todo</li>
</ul>
